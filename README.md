# 📈 Stock Market AI Predictor Dashboard

A real-time stock analysis and prediction tool built with Python and Machine Learning. This application uses historical data to forecast price movements and provide technical buy/sell signals.

## ✨ Key Features
- AI-Powered Predictions: Uses a Random Forest Regressor to forecast the next day's closing price.
- Real-Time Data: Fetches live market data via the Yahoo Finance API (yfinance).
- Technical Indicators: Calculates and visualizes RSI (Relative Strength Index) and 20-day SMA (Simple Moving Average).
- Interactive Visuals: Dynamic, hoverable charts built with Plotly.
- Trading Signals: Automated "Buy," "Sell," or "Hold" recommendations based on momentum analysis.

## 🛠️ Technology Stack
- Language: Python
- AI/ML: Scikit-Learn (Random Forest)
- Data: Pandas, yfinance, ta (Technical Analysis Library)
- Frontend: Streamlit
- Visualization: Plotly

## 📦 How to Run Locally
1. Clone this repo: git clone https://github.com/famouspsd/Stock_AI_Dashboard.git
2. Install dependencies: pip install -r requirements.txt
3. Launch the app: streamlit run app.py
